# Windows-Form-Calculator
A simple calculator application built using **C# and Windows Forms (.NET)**.
This application performs basic arithmetic operations with a user-friendly graphical interface.

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔄 Clear and Backspace functionality
* 🖥️ Simple and responsive UI

---

## 🛠️ Technologies Used

* C#
* .NET Framework / .NET
* Windows Forms

---

## 🚀 How to Run

1. Open the project in **Visual Studio**
2. Build the solution (**Ctrl + Shift + B**)
3. Run the application (**F5**)

---

## 📂 Project Structure

* `Form1.cs` → Main logic of calculator
* `Form1.Designer.cs` → UI design
* `Program.cs` → Entry point of application
* `Calculator.csproj` → Project configuration

---

## 📷 Output

Calculator interface with buttons for numbers and operations.
